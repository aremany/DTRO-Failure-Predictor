# 독립형 장애 예측기 사용 설명서

## 1. 개요
이 프로그램은 과거 장애 이력 데이터를 분석하여 미래의 장애 발생 추이를 예측하는 독립 실행형 도구입니다. GRU(딥러닝), KNN, 선형회귀 등 다양한 알고리즘을 자동으로 적용하여 최적의 예측 결과를 제공합니다.

## 2. 설치 및 실행 방법

### 2.1. 필수 요구 사항
- Python 3.8 이상
- `incident_reports.db` 파일 (상위 폴더 또는 현재 폴더에 위치해야 함)

### 2.2. 설치
1. 명령 프롬프트(CMD) 또는 터미널을 엽니다.
2. 프로그램 폴더로 이동합니다.
   ```bash
   cd "d:\01 공통\공사중\independent_predictor"
   ```
3. 필요한 라이브러리를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```

### 2.3. 실행
1. 다음 명령어로 프로그램을 실행합니다.
   ```bash
   python main.py
   ```
2. 실행 후 웹 브라우저가 자동으로 열리지 않으면, 브라우저 주소창에 `http://localhost:8001`을 입력하여 접속합니다.

## 3. 사용 방법
1. **단일 장애 예측**:
   - 화면 상단의 드롭다운 메뉴에서 장애 유형을 선택하고 `단일 예측` 버튼을 클릭합니다.
   - 결과 카드에서 차트와 상세 정보를 확인합니다.

2. **전체 장애 예측 대시보드**:
   - `🚀 전체 장애 예측 대시보드 실행` 버튼을 클릭합니다.
   - 모든 장애 유형에 대한 예측이 수행되며, 상위 5개 장애의 추세가 차트로 표시됩니다.
   - 하단 테이블에서 전체 순위를 확인할 수 있습니다.

3. **엑셀 저장**:
   - 전체 예측이 완료된 후, 대시보드 우측 상단의 `💾 엑셀 저장 (.xlsx)` 버튼을 클릭합니다.
   - `prediction_report_YYYYMMDD_HHMMSS.xlsx` 파일이 다운로드됩니다.

## 4. 문제 해결
- **DB 연결 오류**: `incident_reports.db` 파일이 프로그램 상위 폴더(`..`)나 현재 폴더에 있는지 확인하세요.
- **엑셀 저장 오류**: `openpyxl` 라이브러리가 설치되어 있는지 확인하세요. (`pip install openpyxl`)
